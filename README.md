#  AI Cybersecurity Assistant

##  Overview
An AI-powered chatbot that detects phishing messages and provides cybersecurity advice.

##  Features
- Phishing detection
- Risk classification (High/Medium/Low)
- Safety suggestions

##  Tech Stack
- Python
- OpenAI API


## Example
Input: "You won a lottery! Click now!"
Output:
Risk Level is HIGH don't click on the message. 
Reason: Message is suspicious don't click.
Advice: Do not click links.
